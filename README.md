# TIC TIC TOE
---
Tic-tac-toe implemented in C.

## Compilation

Use the [make](https://www.gnu.org/software/make/manual/make.html#toc-Overview-of-make) utility to compile the game.

```bash
~$ make tictactoe
```

## Run the game

```bash
~$ ./tictactoe
```
